# Lab 1 (18 - 10 - 2025)

## Git Environment Setup

- Update the system file
    - sudo apt update -y
    - sudo apt upgrade -y
- Check git available or not
    - git —version
    - if not available install git using below command
        - sudo apt install git-all -y
- Check git version again
- Check global configuration of Git
    - git config —global—list
- Configure global credentials:
    - git config —global [user.name](http://user.name) “*give your git username*”
    - git config —global [user.email](http://user.email) “*give your git mail id*”
- Now check your global credentials again using:
    - git config —list
    
    ---
    
- Create a directory and initialize the local repository
    - mkdir my_first_project
    - cd my_first_project
    - git init
- Create a sample file and add some text into it:
    - echo “*Hello DevOps*” >> sample.txt
    - cat sample.txt
- Stage and commit the file:
    - git add sample.txt
    - git commit -m “*Add your message here*”
